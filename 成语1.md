# 001 爱憎分明  [^001]
- ài zēng fēn míng
- · 爱：喜好。 憎：憎恶，恨。
- 憎恶：憎恨；厌恶。恶wù。
- 分明：界限清楚、明确。
- 爱和恨的立场和态度十分鲜明。
- 在大家的眼里，王阳从来都是个爱憎分明的人。
- 在很长一段时间里，我一直为自己这种爱憎分明的价值观洋洋得意。殊不知“爱憎分明”的背后往往是“非黑即白”，“洋洋得意”的背后往往是“愚蠢无知”。
 - ==殊不知==
- shū bùzhī
- 殊：副词。竟，竟然。
- 竟不知道。
- 他还自以为得计,殊不知这样做反而断送了自己。
# 002 安土重迁  [^002]
- ān tǔ zhòng qiān
- · 安：谓对某种环境、事物感到安适或习惯，习惯于。
- 土：乡土；安土：安于故土。
- 重：慎重；谨慎，引申为不轻易。
- 重迁：谓不轻易迁居。
- 习惯于生活在故土，不愿轻易迁居异地。
- 上千年的从事农业耕作，北方人养成了安土重迁的特点。 人普遍都有“安土重迁”的心理，一旦在某个地方待得久了、熟悉了，便不再想迁徙。
# 003 安如磐石 [^003]
- ān rú pán shí
- 安若磐石 安如盤石
- 安如盘石 安于盘石
- 磐石：大石头。
- 如同磐石一般安稳。
- 形容非常稳固，不可动摇。
- 他是公司的元老，地位安如磐石，无可动摇。
- 教室的门先前无须用力，轻轻一推就可以打开。可现在我使劲儿推了一下，它却纹丝不动；我再用力用身体去顶，它仍安如磐石。
# 004 安步当车  [^004]
- ān bù dàng chē
- 安：徐缓；安详，不慌忙；
- 安步：缓缓步行。
- 以从容的步行当作坐车。
- 形容不慌忙、不著急。
- 也指人能够安守贫贱生活。
- <span style="color: #ff00ff">**晚食以当肉，安步以当车，**
- <span style="color: #ff00ff">**无罪以当贵，清静贞正以自虞。**
- 晚点吃饭权当吃肉，悠闲散步权当乘车，
- 不犯王法权当富贵，清静纯正，自得其乐。
	- 虞：古同娱，安乐。
- 每天一到下午，小明爷爷便一个人安步当车，出门逍遥自在去了。
- 这里离钟楼并不远，我们两个人安步当车，片刻即走到了。
- 老子认为如果天下太平，官场干净，就可以出来做官。如果时运不济，官场贪腐，那就做野外的蓬草，在乡下随风而行，安步当车吧。
# 005 安然无恙 [^005]
- ān rán wú yàng
- 安然：平安，安稳。
- 恙：病。无恙：没有灾祸、疾病之类忧愁的事。
- 形容平平安安地没有遭遇任何损害。
- 看到他安然无恙地回到家里，我们都松了一口气。
- 他遭遇连环车祸，竟安然无恙，毫发无伤。
# 006 黯然神伤 [^006]
- àn rán shén shāng
- 黯然：心情沮丧的样子。
- 神伤：神色忧伤；伤心。
- 形容情绪低沉,内心忧伤。
- 长亭，自古以来就是人们饯别之地，离别总令人黯然神伤，这使得“长亭”这两个字的本身就仿佛带着凄凉萧索之意。
- 奶奶听完后不禁黯然神伤，低下头深深地叹了一口气。
# 007 暴殄天物 [^007]
- bào tiǎn tiān wù
- 暴：殘害，糟蹋；
- 殄：尽，灭绝；
- 天物：指自然生物。
- 原指残害灭绝天生万物。
- 后指任意糟蹋东西，不知爱惜。
- 大多数高架桥下空间都处于空置状态，少部分被用作杂物堆放场，这种较低附加值的空间开发可谓暴殄天物。
- 茶叶买的再贵,不会冲泡也是暴殄天物。
# 008 并行不悖 [^008]
- bìng xíng bù bèi
- 并：一齐。并行：同时实行。
- · 悖：违背，冲突。
- 同时进行，不相冲突。
- 作为一个学生，既要刻苦学习，也要注重休息，要保证二者并行不悖。
- 现代与传统并行不悖，是这座酒店的建筑的特点。
# 009 不省人事  [^009]
- bù xǐng rén shì
- 省：知觉。
- 人事：人的意识的对象。也指事理人情。
- 指昏迷不醒，失去知觉。
- 也指不懂人情世故。
- 当大家找到他时，他已经不省人事了。
- 小明都十四岁了，还这么不省人事。
# 010 病入膏肓  [^010]
- bìng rù gāo huāng
- 膏肓：膏：中医指心尖脂肪，认为是药力达不到的部位。
- 肓：心脏与膈膜之间叫“肓”。中医认为“膏肓”是药力达不到的地方。
- 形容病情十分严重，无法医治。
- 比喻事情到了无法挽救的地步。
- 这个病人已病入膏肓，无力回天了。
# 011 便宜行事   [^011]
- biàn yí xíng shì
- · 便宜：方便合宜，适宜。
- 行事：办事；做事。
- 指不必请示，自行决断处置。
- 在不违反公司政策的状况下，外派人员可以便宜行事。
# 012 比肩接踵  [^012]
- bǐ jiān jiē zhǒng
- 比肩继踵 继：接着。
- 比：挨著；并列。
- 比肩：肩并著肩。
- 形容人很多或接连不断。
# 013 摩肩接踵  [^013]
- mó jiān jiē zhǒng
- 摩：接近；迫近。接：连接，接着。踵：脚后跟。
- 肩碰着肩，脚挨着脚。
-  形容往来人多，拥挤不堪。 
-  公园里，游人摩肩接踵，十分拥挤。 
# 014 接踵而来 [^014]
-  jiē zhǒng ér lái
- 踵：脚后跟。
- 接踵：后面人的脚尖接着前面人的脚后跟。
- 指人们前脚跟着后脚，接连不断地来。
- 形容来者很多，络绎不绝。
-  一连串的事情接踵而来，搞得我焦头烂额。 
# 015 不落窠臼  [^015]
- bù luò kē jiù
- · 窠：鸟巢；臼：舂米的石器。
- 窠臼：指文章或艺术作品所墨守的老格式、旧框框。
- 比喻文章或艺术有独创风格，不落旧套。
-  这篇作文立意新颖，不落窠臼。 
-  伟大的探索家、发明家或是杰出的天才，无论男女，都是那些不落窠臼，敢于打破传统，富于创造的人。
- 窠臼 kējiù
- 窠：鸟巢；臼：舂米的石器。
- 门臼。旧式门上承受转轴的臼形小坑。
- 比喻旧有的现成格式；老套子。
-  14世纪欧洲人开始解放思想，打破窠臼，积极进取，乃至于不择手段地捞钱、发展、图强。 
# 016 不容置喙  [^016]
- bù róng zhì huì
- 置：放，摆，搁。 喙：嘴。
- 指不容许别人插嘴说话。
-  王老师普普通通的一句话，却透出这句话中不容置喙的意味。 
# 017 不胫而走  [^017]
- bù jìng ér zǒu
- 胫：小腿；而：表转折关系。
- 走：快跑。
- 没有腿却能奔跑。
- 形容消息等传播迅速，都不用专门宣传。
-  这个消息无翼而飞，不胫而走，很快就传遍了全校。 
# 018 无翼而飞  [^018]
- wú yì ér fēi
- 翼：翅膀。没有翅膀却飞走了。
- 也比喻事情传播得很迅速。
-  这个消息无翼而飞，不胫而走，很快就传遍了全校。 
# 019 不翼而飞 [^019]
- bù yì ér fēi
- 翼：翅膀。没有翅膀却飞走了。
- 比喻物品忽然丢失。
- 也比喻事情传播得很迅速。
-  回到教室的小明发现，自己放在书包里的钱不翼而飞了，在班上引起了极大的不安。 
# 020 惨绝人寰  [^020] 
- cǎn jué rén huán
- 绝：穷尽。 惨绝：惨痛之极。
- 人寰：人世间。寰：寰宇（全世界）。
- 人世间再没有比这更凄惨残酷的了。
- 形容惨痛到了极点。
- 也形容狠毒、残暴到极点。
-  在惨绝人寰的南京大屠杀中，30万手无寸铁的平民被屠杀。 
-  一五一年洛阳旱灾，翌年地震，继之蝗灾、水患，人民饥饿难当，互相吞食以苟活，情境惨绝人寰。 
# 021 惩前毖后   [^021]
chéng qián bì hòu
- ==前惩后毖==
- 惩：警戒，戒止；毖：谨慎。
- 把以往的错误作为教训，使以后谨慎行事，不致再犯。
-  对犯错误的人，我们应该采取惩前毖后、治病救人的态度。 
-  惩罚不是手段，理解沟通才是手段，我们经常说的惩前毖后、治病救人，就这个道理。 
- 欧美国家发展的经验教训，不论对哪个国家而言，不仅具有惩前毖后的时代意义，更具有警醒和启示的现实意义。 
# 022 长吁短叹  [^022]
- cháng xū duǎn tàn
- 吁：叹气。
- 长一声、短一声不住地叹气。
- 形容发愁的神情。
-  最近老是见他一个人坐在那长吁短叹的。 
# 023  缠绵悱恻  [^023]
- chán mián fěi cè
- 缠绵：情意深厚，纠缠不已，不能解脱。
- 悱恻：内心悲苦的样子。
- 形容心情悲苦，情感纠结，难以排解。
- 也指文章，音乐等哀转凄切。
-  缠绵悱恻的爱情故事是小说中常见的题材。 
-  李义山的诗写得总是缠绵悱恻，动人至深。 
# 024 叱咤风云  [^024]
- chì zhà fēng yún
- 叱咤：怒喝声。
- 一声呼喊、怒喝，可以使风云翻腾起来。
- 形容威力极大。
-  这位老前辈曾是叱咤风云的人物。 
- 短短几年，以前默默无闻的这几个人，如今都成为了叱咤风云的行业巨头。 
- 中华文明璀璨夺目，在这片古老的华夏大地上，涌现出一个个叱咤风云的历史人物，留下了许多可歌可泣的丰功伟绩。 
# 025 参差不齐  [^025]
- cēn cī bù qí
- 参差：不齐，杂乱不整齐的样子。
- 形容水平不一或很不整齐。
-  同学们的学习成绩参差不齐。 
-  二十四史中，水平参差不齐，其中哪一部也不能望史记之项背。 
-  由于同学们的情况参差不齐，其中有很多人毕业已经好几年了，连单词都认不得几个。 
# 026 瞠目结舌  [^026] 
- chēng mù jié shé
- 瞠：瞪着眼；
- 结舌：说不出话来。瞪着眼睛说不出话来。
- 形容窘困或惊呆了的样子。
- 窘困：处境为难或难堪。
-  他的这一番举动，让人瞠目结舌。 
-  面对老师突然提出的问题，全班同学个个瞠目结舌，无言以对。 
# 027 蹉跎岁月  [^027] 
- cuō tuó suì yuè
- 蹉跎：光阴白白的过去。　岁月：年月日子;时间。
- 形容虚度光阴。
-  努力付出者必将收获美梦成真的喜悦；蹉跎岁月者换回的只是时间在脸上留下的悔恨，最终一事无成。 
-  我们每个人都应该努力地学习，否则，蹉跎岁月，韶华易逝，当错过了宝贵的学生时代，就悔之晚矣。 
- 韶华：韶，美好的。华：时光。
- 美好的时光。常指春光。常比喻美好的青年时光。
# 028 垂涎三尺  [^028] 
- chuí xián sān chǐ
- 涎：口水。口水挂下三尺长。
- 形容非常嘴馋或贪恋他人东西，想据为己有。
-  两碗汤端上桌，最底下是一只被煮得发白的猪蹄，嫩嫩的肉白花花地露了出来，让人禁不住垂涎三尺。 
-  陶是当时天下最富庶、最发达的地方，不仅齐国人对陶垂涎三尺，全天下的人都对这块地盘想入非非。 
# 029 嗤之以鼻  [^029]
- chī zhī yǐ bí
- 嗤：讥笑。之：别人。
- 用鼻子发出冷笑声，讥笑别人。
- 表示鄙视、看不起。
-  他的卑劣行径令人嗤之以鼻。 
-  面对小明的质问，小红却是嗤之以鼻，“我早受够了你一副施舍的姿态了。 
# 030 大腹便便  [^030] 
- dà fù pián pián
- 便便：肥胖的样子。
- 形容肚子肥大的样子。
-  这个秃顶的男人五十上下，大腹便便，水汪汪的蓝眼睛上架着一副老式眼镜。 
-  我们没等太久，一位穿深色衣服大腹便便的人便出来迎接我们了。 
# 031 殚精竭虑  [^031] 
- dān jīng jié lǜ
- 殚：耗尽； 精：精力。
- 竭：竭尽； 虑：思虑。
- 形容用尽心思。
-  塔塔的工程师和设计师们殚精竭虑，苦心钻研，耗费了四年时间,终于实现了这个目标。 
-  钱学森老人走了，这个为中国科技事业殚精竭虑了一生的科学家，在2009年北京飘起第一场大雪前走了，但他伟大的一生并未结束，他留给世人无穷尽的财富。 
# 032 丢卒保车  [^032]
- diū zú bǎo jū
- 原是象棋术语。
- 后比喻丢掉次要的，保住主要的。
-  我们也应该正确看待失去，学会从容地接受失去。为了为了更好地实现自己的目标和理想，有时不得不丢卒保车。
- 为了不让老师发现手机，他也只得丢卒保车，主动把零售都交了出来。 
# 033 度德量力  [^033] 
- duó dé liàng lì
- 度：估量；德：德行。
-  衡量自己的德行是否能够服人，估计自己的能力是否能够胜任。
- 做任何事情都不能好高婺远，首先要掂量自己的水平，度德量力，量力而行才行。 
# 034 大煞风景  [^034] 
- dà shā fēng jǐng
- 大杀风景 大煞风趣
- 损伤美好的景致。
- 比喻败坏兴致。
-  这么澄澈的湖面上竟漂著一个垃圾袋，真是大煞风景。
- 这栋楼设计的奇丑无比，矗立在十字路口的位置，很是碍眼，有点大煞风景。 
# 035 胆战心惊  [^035]
- dǎn zhàn xīn jīng
- 胆颤心惊
- 胆：犹言肝胆；心曲。
- 心曲： 内心深处。
- 战：通“颤”，发抖。
- 惊：害怕，精神受了突然刺激而紧张不安。
- 形容心里十分害怕。
-  这幻象突然站在这里了，看得见摸得着，非常真切，令人不寒而栗、毛骨悚然、胆颤心惊。
- 我走向玻璃扶手，全透明的围栏让人有点胆颤心惊。空气很新鲜，但风有点凉，因为我站的地方是如此高。 
# 036 独占鳌头  [^036] 
- dú zhàn áo tóu
- 鳌头
- 宫殿门前台阶上的鳌鱼浮雕，科举进士发榜时状元站此迎榜。科举时代指点状元。
- 比喻占首位或第一名。
-  期中考试，小明再一次独占鳌头，而我的成绩在平时学得好的几个人中，差不多算垫底了，这中间的差距让我很焦虑。
 - 除极少数怯懦者，人们无一不想成为生活中的佼佼者，无一不想在各自所从事的领域里独占鳌头，就算学校里的小学生也希望自己排名全班第一。 
-  桂冠
- 古希腊常用月桂树叶编织成的帽子授予杰出的诗人或竞技比赛的优胜者，象征光荣。后借“桂冠”代替第一。
- 问鼎鼎是古代国家政权的象征。敢于“问鼎”者，说明力量很强大，有能力掌管大事，即“第一者”。
- 榜首古代科举考试后，列名张榜，“榜首”即列于全榜之上的第一名。
- 夺魁“魁”指为首者，“夺魁”就是“夺取第一”。
-  执牛耳古代诸侯订立盟约时，立盟者要亲手宰割牛耳取血，让每人尝一口血。后来便以“执牛耳”象征在某一方面居领导地位，意为“第一”。 
# 037 咄咄逼人 [^037]  
- duō duō bī rén
- 咄咄：表惊诧。也用以表示感叹。
- 逼人：给人造成威胁或压力。
- 原指说话刺人，使人难堪。
- 形容气势汹汹，盛气凌人。
- 也形容后辈超越前人，令人赞叹。
- 也指形势发展迅速，给人压力。
-  他说话的口气咄咄逼人，令人十分难堪。
- 面对老师咄咄逼人的问话，只见他频频拭汗，竟说不出半个字来。 
# 038 独辟蹊径 [^038]  
- dú pì xī jìng
- 蹊径：小路或山路。“蹊”与“径”同义，指小路；引申为方法、出路，风格等。
- 独自开辟一条与众不同的道路。
- 比喻独创一种风格或新的方法。
-  从创建伊始，国家大剧院就认识到西洋歌剧在中国几乎空白的现状，他们独辟蹊径，已经并正在做着诸多开创性的尝试。
- 在学习中他往往喜欢独辟蹊径，有着自己独特的一套学习方法。 
# 039 海市蜃楼 [^039] 
- hǎi shì shèn lóu
- 海市：大气因光折射而形成的反映地面物体的形象。旧称蜃气。
-  光线因不同密度空气层的折射，把远处景物显示在空中或地面的奇异幻景。多发生在夏天沿海一带或沙漠中，发生时可见极远处隐约有山峦、城池、亭台楼阁等，不久就全部消失。古人误以为由蜃：大蛤蜊gé lí 吐气所形成的，称之为蜃楼，也叫海市。 
- 现多比喻虚无缥渺的事物。
-  梦想不是空想，不是海市蜃楼，实现梦想需要实实在在的行动。
- 她的梦想不过是自欺欺人的海市蜃楼。
# 040 户枢不蠹 [^040]
-  hù shū bù dù
- 户：门。枢：门轴。蠹：虫蛀。
- 经常转动的门轴不会被虫蛀。
- 比喻经常运动的东西不容易受侵蚀。
- 也比喻人经常运动可以强身。
-  活动能帮助消化，促进血液流通，不易发生疾病，这也就是先哲思想里“户枢不蠹”的道理。
- 人的生命在于运动，所谓"流水不腐，户枢不蠹"，因此要保持健康的身体，就应该使五脏六腑、肌肤、血脉经常受到锻炼。
# 041 好逸恶劳  [^041]
- hào yì wù láo
- 逸：安逸；恶：讨厌、憎恨。
- 贪图安逸，厌恶劳动。
-  由于父母从小就溺爱他，让他养成了好逸恶劳的坏习惯。
- 好逸恶劳本是人的通性，所以陶侃运甓pì，祖逖闻鸡起舞，方始被人称颂。
# 042 混淆是非  [^042] 
- hùn xiáo shì fēi
- 混淆：使界限不清。
- 故意把正确的说成错误的，把错误的说成正确的。
-  他们几个人故意颠倒黑白，混淆是非。 
-  大家都见识过小明睁眼说瞎话，谁能保证他这次不会又是歪曲事实、混淆是非呢？ 
-  如果东西不是小明拿的，而非要说成是他拿的，这种混淆是非的事情我干不出来。 
# 043 汗流浃背 [^043]  
- hàn liú jiā bèi
- 浃：湿透。 背：脊背。
- 流的汗湿透了背上的衣服。
- 形容由于天热、劳累、恐惧、惭愧等而出汗很多。
-  比赛结束后，选手们个个汗流浃背。 
-  临近黄昏，天气热得令人窒息。我们不停拿水壶喝水，尽量一动不动，可还是满头大汗，汗流浃背。 
# 044 呼天抢地  [^044]
- hū tiān qiāng dì
- 呼天：指向天呼号。形容极端痛苦。
- 抢：撞，触。抢地：用头撞地。
- 大声向天呼号，用头撞地。
- 形容极度悲伤，大哭的样子。
-  这女人呼天抢地，痛不欲生，过往行人无不同情。 
-  老师开口才刚这么一问，小明就呼天抢地，大喊冤枉。 
# 045 怙恶不悛  [^045] 
- hù è bù quān
- 怙：坚持。悛：悔改。
- 坚持作恶，不肯悔改。
-  倘若一个人怙恶不悛、不知悔改，那么就算是立下再多痛改前非的誓言又有何用？ 
-  他们三个都是怙恶不悛、不知悔改的家伙，对他们来讲，没有什么事情不可以干的。 
# 046 讳疾忌医   [^046] 
- huì jí jì yī
- 讳：忌讳，不能说，不能碰，不能做，隐藏起来。
- 忌：认为不适宜而避免。
- 隐瞒病情，不愿医治。
- 比喻怕人批评而掩饰自己的的缺点和错误。
-  有病就要看医生，讳疾忌医只会让病情加重。 
-  犯了错误，是知错就改，还是强词夺理、讳疾忌医？我们应该学会如何选择。 
# 047 火中取栗  [^047] 
- huǒ zhōng qǔ lì
- 比喻被人利用，担了风险，吃了苦头。却没有捞到任何好处。
- 出自法国作家拉·封登的寓言。说是炉中烤着栗子。猴子叫猫去偷，猫不但没有吃到栗子，反而把脚上的毛烧掉了。
-  不止一次被人利用了的小明，发誓从今以后，他再也不做替人火中取栗的事了。 
- 我冷笑道：“如果我没理解错的话，你是想让我为你这个家伙火中取栗？” 
# 048 沆瀣一气  [^048] 
- hàng xiè yī qì
- 沆瀣：夜间的水气，露水。这里指唐时的崔沆、崔瀣。
- 一气：指声气相通，结为一伙。
- 比喻臭味相投的人结合在一起。
-  这两个人，一个尖酸，一个刻薄，沆瀣一气，到处惹是生非。 
-  他们互相勾结，沆瀣一气，暗地里大搞阴谋活动。 
# 049 好高骛远  [^049]
- hào gāo wù yuǎn
- 好：喜欢；骛：追求。
- 比喻不切实际地追求过高过远的目标。
-  在学习上要脚踏实地，循序渐进，不能好高骛远。 
-  对于任何一个人来说，不怕他的追求有多么千奇百怪，甚至是好高骛远，就怕没有任何追求。 
# 050 混为一谈 [^050]   
- hùn wéi yī tán
- 把不同的事物混在一起，当作同样的事物谈论。
-  这两件事虽有关联，却不可混为一谈，弄错主次。 
-  世界上叫几内亚的国家有好几个，很多人不熟悉它们的区别，常被混为一谈。 
-  多少年来，人们对比兴的定义却一直纠缠不清。有的学者把比兴混为一谈，有的又主张截然分开。其实，比和兴经常是相互配合地以互补的面貌出现在诗歌中的。
